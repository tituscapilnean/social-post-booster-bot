### Post (posted hook 3)
One in five packages in the OpenClaw MCP registry is malicious. Anthropic's tokenizer swap just made running the rest of them more expensive.

Anthropic shipped a new tokenizer in Opus 4.7. Sticker price unchanged, but the same input now maps to 1.0 to 1.35 times more tokens per call. Nathan Lambert pointed out the quieter truth: a new tokenizer means a new base model. Nate Jones ran four days of production testing and called it the tokenizer tax. GitHub Copilot is the canary. They paused Pro and Pro+ signups, pulled Opus from the Pro tier, and internal docs show weekly Copilot costs nearly doubled since January. The platform with the deepest usage data looked at the new math and flinched.

Meanwhile, Antiy CERT flagged 1,184 malicious skills on ClawHub, roughly one in five packages in the OpenClaw registry at peak. Analysis of 2,614 MCP implementations found 82 percent use file operations exposed to path traversal and 67 percent use APIs prone to code injection. Shopify just gave MCP write access to 5.6 million stores. That is the stack every agent team is about to ship on.

Most builders are treating model selection and MCP security as separate problems. They are the same problem. A team now paying 1.3 times more per Opus call whose agent can hit a hostile tool is double-taxed: higher cost per unit of work, and a widening blast radius for every wrong tool call.

The move that compounds is routing. Moonshot's Kimi K2.6 shipped this week 8 to 10 times cheaper than Opus for agentic work, open weights, 262k context. Anthropic's own Advisor tool shows Sonnet plus Opus tiered reasoning hits 74.8 percent on SWE-bench at 11.9 percent lower cost than solo Sonnet. The frontier is not a single model. It is the routing layer between models and the gateway between your agent and a supply chain you cannot trust by default.

2026 is the year margin stops living in the model and starts living in the infra around it.
