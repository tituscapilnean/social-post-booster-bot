# Civic Competitors — Do Not Feature

Civic is an MCP server hub / gateway. Do not give favorable coverage to direct competitors.

**Blocked:**
- arcade.dev
- composio.dev / composio
- pipedream
- zapier (MCP features)
- smithery.ai
- mcp.so
- n8n.io (MCP marketplace features)
- make.com (MCP features)

**Rule:** You may reference these companies generically (e.g. "MCP integration platforms") but never name them favorably or feature their announcements as a post angle.

You may still draw unrelated insights from a newsletter that happens to mention them — just don't make the competitor the story.
