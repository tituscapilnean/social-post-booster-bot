**Taste Matters in Agents** — Peter Steinberger explained that AI agents still produce "slop" because they lack human taste and direction. While highly capable, they depend on iterative guidance — each prompt shaped by ongoing feedback and judgment. He warned that relying on a single upfront specification and stepping away too early breaks the human–AI loop, leading to mediocre, unfocused results despite strong underlying model capabilities.

**OpenClaw Runs Vending Machine** — Christian Van Der Henst built an OpenClaw-powered vending machine inside a San Francisco startup tower. The AI agent manages the business end-to-end: choosing products, setting prices, naming items, running ads, tracking revenue. Humans handle only physical restocking.

**OpenClaws Start Following Humans** — Community demo with Intel RealSense camera + robot enables real-time person tracking. Vision input + agent decision-making, early physical-world autonomy beyond chat.

**Security vs Workflow Pain** — OpenClaw's update/security improvements repeatedly breaking existing workflows; devs manually re-enable blocked CLI tools (gog, gh) after each update. Peter acknowledged the tension between safety and developer experience.

**OpenClaw 4.12 shipped** — Stability/usability fixes across chat, memory, channels, execution. Local model (LM Studio) and Codex integration improvements.

**Build Your Own OpenClaw** — Open-source 1k-star project walks devs through building an OpenClaw agent from scratch in 18 steps: chat loop → multi-agent routing → event-driven architecture → scheduling → concurrency control → long-term memory.
