**Thesis: OpenClaw is the layer people build entire workflows, businesses, and communities around.**

**Peter Steinberger brings OpenClaw to TED.** Talk traces his childhood obsession with building things to the moment he first let an AI agent loose on the internet. Pitch: agents aren't better chatbots — they're a major shift in how people work, build, create.

**Skill theft hits ClawHub.** A developer found someone copied his open-source GitHub skill `baoyu-diagram` and published it to ClawHub under another account before he could. ClawHub uses unique slugs — he was locked out of publishing under its real name. Peter initially defended first-come-first-serve, then agreed direct copies using someone else's creator prefix should be handled differently; fix coming.

**Y Combinator CEO Garry Tan launched GBrain v0.11 with "Minions" system.** Purpose: fix subagent timeouts and failures. Built a queue + jobs layer on Postgres, PGLite, and BullMQ. Claims ~10x faster. Plans to submit back to OpenClaw as PR.

**OpenClaw cost audit:** Developer cut $680/mo setup to ~$225 without losing capability. Six steps: trim bloated workspace files, disable unused tools, replace many MCP connections with Composio, split work across focused agents, use sub-agents for heavy tasks, run `/context detail` regularly.

**X gets more agent-friendly.**
- X cut cost of reading owned data through its API 5x to $0.001. Posts, bookmarks, mentions, lists far cheaper to access.
- Raised posting costs and limited growth actions (follows, quote-posts).
- Elon Musk encouraged devs to use OpenClaw to build apps on top of X. Platform explicitly wants more agents, less spam automation.

**Community take:** OpenClaw did more than hype agents. For many, it was the first real introduction to agent workflows, memory systems, local-first setups, multi-tool orchestration, and open-source agent building.

**Takeaway: the agent stack is growing up in public.** Skill marketplaces need governance. Subagent infrastructure needs production-grade queues. Cost audits are becoming routine ops. And platforms like X are actively tuning pricing to attract (or filter) agent traffic.
